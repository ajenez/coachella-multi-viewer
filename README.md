# Coachella 2026 Multi-Viewer

Watch all 7 Coachella stage streams at once — with live schedules, a full-screen main player, and a visual timeline. No install, no account, no setup. Just open the link.

🔗 **[ajenez.github.io/coachella-multi-viewer](https://ajenez.github.io/coachella-multi-viewer)**

## Features

- **7 simultaneous streams** — Coachella Stage, Outdoor Theatre, Sahara, Mojave, Gobi, Sonora, Quasar
- **Live schedule** — Current artist, set progress bar, and upcoming acts on every card
- **Main view** — Full-screen focused player with instant stage switching
- **Stage switcher** — Swipeable thumbnail row for quick crossfade switching
- **Full schedule overlay** — Visual timeline across all stages for all 6 festival days
- **Theater mode** — Hides the UI for a distraction-free view
- **Setlist tracker** — Shows songs played so far via setlist.fm (where available)
- **Rebroadcast indicator** — Flags when a stage is replaying earlier content
- **All times in PDT** — Festival timezone throughout

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `1` – `7` | Switch to stage 1–7 |
| `M` | Mute / unmute |
| `T` | Toggle theater mode |
| `S` | Open / close schedule |
| `Esc` | Go back |
